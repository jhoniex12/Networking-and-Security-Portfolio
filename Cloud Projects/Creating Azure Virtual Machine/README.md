# Creating an Azure Virtual Machine

A virtual machine (VM) is basically a computer inside another computer. It is not physical but will behave as if you have a real computer with its own operating system, CPU, memory, storage, and network. This said computer is running in a data center in which they run it 24/7 to give you uninterrupted service. In Microsoft Azure, a VM is an Infrastructure as a Service (IaaS). It Instead of using physical hardware directly, a VM uses virtualized resources provided by a host machine or cloud platform like Microsoft Azure.

In Microsoft Azure, VMs are part of IaaS, where users can create and manage virtual systems easily. They are commonly used for testing, development, hosting apps, and remote access.
